# Stellies iCafe - Digital Upgrade (Stellenbosch, SA)
**Live Demo:** [https://stellies-i-cafe.vercel.app](https://stellies-i-cafe.vercel.app)

## 🎯 Project Overview
This project is an independent redesign and functional upgrade for **Stellies iCafe**, a local tech and service hub in Stellenbosch. The goal was to transform a dated web presence into a high-performance, mobile-first **Single Page Application (SPA)** that better serves the local student and professional community.

### 🔄 The Transformation
* **The 'Before':** A static, non-responsive site that lacked interactive customer tools.
* **The 'After':** A modern "Glassmorphism" interface with real-time business logic and full Dark Mode support.

## 🚀 Key Business Features
* **ZAR Price Calculator:** Integrated logic to help customers calculate printing, scanning, and ID photo costs in Rands before visiting the shop.
* **Live Status Indicator:** A JavaScript-powered badge that checks South African Standard Time (SAST) to show if the shop is 'Open Now' or 'Closed'.
* **Conversion-First Design:** Direct "One-Tap" integration for WhatsApp, Phone, and Google Maps (Libertas Building).
* **Performance:** Optimized for instant loading on mobile data (built with Vite).

## 🛠️ Technical Stack
* **Frontend:** React 18
* **Styling:** Tailwind CSS
* **Language:** TypeScript (for production-grade stability)
* **Deployment:** Vercel (CI/CD)

## 👨‍💻 Development Journey
